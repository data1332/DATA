🔗 اطلاعات API

این صفحه به عنوان API برای اپلیکیشن دیکشنری عمل می‌کند.

📡 آدرس API:

https://your-website.com/dictionary_api.html

📋 فرمت پاسخ:

Content-Type: application/json

📱 نحوه استفاده در اپلیکیشن:

  1. این فایل را روی یک وب‌سرور آپلود کنید
  2. آدرس کامل فایل را کپی کنید
  3. در اپلیکیشن، دکمه "تنظیمات" را بزنید
  4. آدرس API را وارد کنید
  5. دکمه "ذخیره" را بزنید
  6. دکمه "بروزرسانی داده‌ها" را بزنید

📊 پیش‌نمایش داده‌ها

داده‌های زیر در فرمت JSON برای اپلیکیشن ارسال می‌شوند:

React
دسته‌بندی: برنامه‌نویسی
یک کتابخانه جاوااسکریپت برای ساخت رابط کاربری تعاملی و واکنش‌گرا. React توسط فیسبوک توسعه داده شده و برای ساخت اپلیکیشن‌های تک صفحه‌ای (SPA) استفاده می‌شود.
TypeScript
دسته‌بندی: برنامه‌نویسی
نسخه‌ای از جاوااسکریپت با تایپ قوی‌تر و ویژگی‌های پیشرفته. TypeScript توسط مایکروسافت توسعه داده شده و به توسعه‌دهندگان کمک می‌کند تا کدهای قابل اعتمادتری بنویسند.
Tailwind CSS
دسته‌بندی: طراحی
فریم‌ورک CSS برای طراحی سریع و واکنش‌گرا با کلاس‌های آماده. Tailwind CSS به توسعه‌دهندگان اجازه می‌دهد تا بدون نوشتن CSS سفارشی، رابط‌های کاربری زیبا بسازند.
Node.js
دسته‌بندی: برنامه‌نویسی
محیط اجرای جاوااسکریپت برای سمت سرور و توسعه اپلیکیشن‌ها. Node.js به توسعه‌دهندگان اجازه می‌دهد تا با استفاده از جاوااسکریپت، اپلیکیشن‌های سمت سرور بسازند.
API
دسته‌بندی: تکنولوژی
رابط برنامه‌نویسی که به اپلیکیشن‌ها اجازه ارتباط با هم می‌دهد. APIها به عنوان پل ارتباطی بین سرویس‌های مختلف عمل می‌کنند و امکان تبادل داده را فراهم می‌آورند.
